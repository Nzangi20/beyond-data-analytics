3.1 Chi-Square Goodness-of-Fit Test

The Chi-Square Goodness-of-Fit test determines whether a single categorical variable follows a specified theoretical distribution. It compares the observed frequencies in your sample to the expected frequencies if the null hypothesis were true.

The Hypotheses
  • Null Hypothesis (H0): The observed frequencies match the expected frequencies (the data follows the theoretical distribution).
  • Alternative Hypothesis (HA): The observed frequencies do not match the expected frequencies.
The Test Statistic

The Chi-Square statistic measures the squared deviation between observed (O) and expected (E) counts, scaled by the expected count:

χ2 = Σ [ (Oi - Ei)2 / Ei ]

Where df = k - 1 (k is the number of categories).

Clinical Example: Genetic Ratios

Suppose a genetic theory predicts that a certain trait should appear in a 3:1 ratio (Dominant:Recessive). In a sample of 100 patients, you observe 78 Dominant and 22 Recessive. Does the data fit the theory?

  • Expected: 75 Dominant, 25 Recessive.
  • Calculation: χ2 = [(78-75)2/75] + [(22-25)2/25] = 0.12 + 0.36 = 0.48.
  • Conclusion: With df = 1, the critical value at α=0.05 is 3.84. Since 0.48 < 3.84, we fail to reject H0. The data fits the 3:1 genetic model.
R Implementation
# Observed counts
observed <- c(Dominant = 78, Recessive = 22)

# Expected probabilities (3:1 ratio)
expected_probs <- c(0.75, 0.25)

# Perform Chi-Square Goodness-of-Fit Test
chisq.test(observed, p = expected_probs)

3.2 Chi-Square Test of Independence

This test evaluates whether there is a significant association between two categorical variables. It uses a contingency table (cross-tabulation) to compare observed joint frequencies against expected frequencies assuming the variables are independent.

Calculating Expected Frequencies

If the two variables are truly independent, the expected count for any cell in the contingency table is calculated as:

Eij = (Row Totali × Column Totalj) / Grand Total
Degrees of Freedom

For a table with r rows and c columns:

df = (r - 1) × (c - 1)
Clinical Example: Thalassemia Stress Test vs. Heart Disease

Using the Heart.csv dataset, we want to know if the Thal result (Normal, Fixed Defect, Reversible Defect) is independent of AHD (No Disease, Disease).

Interpretation Rule of Thumb: If the p-value < 0.05, we reject the null hypothesis of independence and conclude that the two variables are associated. In clinical terms, the stress test result provides diagnostic information about the presence of heart disease.
R Implementation
# Load data
heart <- read.csv("Heart.csv")
heart <- na.omit(heart) # Remove missing values for Chi-Square

# Create contingency table
tab <- table(heart$Thal, heart$AHD)
print(tab)

# Perform Chi-Square Test of Independence
chi_test <- chisq.test(tab)
print(chi_test)

# Extract Expected Counts to verify assumptions
print(chi_test$expected)
Python Implementation
import pandas as pd
from scipy.stats import chi2_contingency

# Load and clean data
df = pd.read_csv("Heart.csv").dropna(subset=['Thal', 'AHD'])

# Create contingency table
contingency_table = pd.crosstab(df['Thal'], df['AHD'])

# Perform Chi-Square Test
chi2, p, dof, expected = chi2_contingency(contingency_table)

print(f"Chi2 Statistic: {chi2:.4f}")
print(f"P-value: {p:.6f}")
print(f"Degrees of Freedom: {dof}")

3.3 Fisher's Exact Test

The Chi-Square test is an approximation that relies on large sample sizes. A common rule of thumb is that the Chi-Square test is invalid if more than 20% of the cells have expected frequencies less than 5. In these cases, we must use Fisher's Exact Test.

When to Use Fisher's Exact Test
  • Small sample sizes (e.g., n < 20).
  • Highly imbalanced contingency tables (e.g., a rare disease vs. a rare exposure).
  • Any cell in a 2x2 table has an expected count < 5.
How It Works

Unlike Chi-Square, Fisher's Exact Test does not rely on an approximation. It calculates the exact probability of observing the specific arrangement of data in the table, assuming the null hypothesis of independence is true, using the Hypergeometric Distribution.

p = [ (a+b)! (c+d)! (a+c)! (b+d)! ] / [ a! b! c! d! n! ]

Where a, b, c, d are the counts in the four cells of a 2x2 table, and n is the grand total.

R Implementation
# Example: Rare side effect of a new drug
# Rows: Drug vs Placebo
# Cols: Side Effect vs No Side Effect
rare_table <- matrix(c(2, 8, 1, 19), nrow = 2, 
                     dimnames = list(Treatment = c("Drug", "Placebo"), 
                                     Outcome = c("SideEffect", "NoSideEffect")))

# Perform Fisher's Exact Test
fisher.test(rare_table)

3.4 Introduction to Analysis of Variance (ANOVA)

ANOVA is a statistical method used to test differences between two or more means. It was developed by Ronald Fisher to solve a critical problem: Why not just use multiple t-tests?

The Problem with Multiple t-Tests

If you want to compare the mean cholesterol levels of 4 different diet plans, you would need to run 6 separate t-tests (Plan 1 vs 2, 1 vs 3, 1 vs 4, 2 vs 3, 2 vs 4, 3 vs 4). If you set your significance level (α) to 0.05 for each test, the probability of making at least one Type I error (False Positive) across all 6 tests skyrockets to nearly 26%! This is called the Family-Wise Error Rate (FWER) inflation.

The ANOVA Solution

ANOVA tests the global null hypothesis that all group means are equal in a single step, protecting the overall α level.

  • H0: μ1 = μ2 = ... = μk
  • HA: At least one mean is different from the others.
The F-Statistic: Partitioning Variance

ANOVA works by partitioning the total variability in the data into two components:

  1. Between-Group Variance (SSB): Variability due to the differences between the treatment groups (the "signal").
  2. Within-Group Variance (SSW): Variability due to random error or individual differences within the same group (the "noise").
SST (Total) = SSB (Between) + SSW (Within)

The F-statistic is the ratio of the Mean Square Between to the Mean Square Within:

F = MSB / MSW = (SSB / dfB) / (SSW / dfW)

If the treatment has a real effect, the Between-Group variance will be much larger than the Within-Group variance, resulting in a large F-value and a small p-value.

3.5 Assumptions of ANOVA

Like all parametric tests, ANOVA relies on strict mathematical assumptions. Violating these assumptions can lead to invalid p-values and incorrect conclusions.

The data points in different groups must be independent. A subject in Group A cannot influence a subject in Group B. Furthermore, there is no "repeated measures" (one subject measured multiple times) in a standard One-Way ANOVA.

How to ensure: Proper randomization and experimental design.

The dependent variable should be approximately normally distributed within each group.

How to check:

  • Visual: Q-Q plots, Histograms.
  • Statistical: Shapiro-Wilk test (H0: Data is normal. If p < 0.05, normality is violated).

Remedy: If violated, apply a data transformation (e.g., Log, Square Root) or use a non-parametric alternative like the Kruskal-Wallis test.

The variance of the dependent variable must be equal across all groups. (σ12 = σ22 = ... = σk2).

How to check:

  • Levene's Test: More robust to departures from normality. (H0: Variances are equal. If p < 0.05, variances are unequal).
  • Bartlett's Test: More powerful but highly sensitive to non-normality.

Remedy: If violated, use Welch's ANOVA, which does not assume equal variances.

3.6 One-Way ANOVA

A One-Way ANOVA is used when you have one categorical independent variable (factor) with three or more levels, and a continuous dependent variable.

Clinical Example: Maximum Heart Rate by Chest Pain Type

We want to know if the mean Maximum Heart Rate (MaxHR) differs across the three types of Chest Pain (ChestPain: typical, nonanginal, asymptomatic).

R Implementation
# Load data and filter for complete cases
heart <- na.omit(read.csv("Heart.csv"))

# 1. Check Assumptions
# Normality (Shapiro-Wilk on residuals)
model_aov <- aov(MaxHR ~ ChestPain, data = heart)
shapiro.test(residuals(model_aov))

# Homogeneity of Variance (Levene's Test)
library(car)
leveneTest(MaxHR ~ ChestPain, data = heart)

# 2. Run One-Way ANOVA
anova_result <- aov(MaxHR ~ ChestPain, data = heart)
summary(anova_result)

# 3. Interpret Output
# Look at the "Pr(>F)" column. If < 0.05, at least one group mean is significantly different.
Python Implementation
import pandas as pd
import scipy.stats as stats
from statsmodels.formula.api import ols
from statsmodels.stats.anova import anova_lm

# Load data
df = pd.read_csv("Heart.csv").dropna()

# 1. Check Homogeneity of Variance (Levene's Test)
groups = [group['MaxHR'].values for name, group in df.groupby('ChestPain')]
stat, p = stats.levene(*groups)
print(f"Levene's Test p-value: {p:.4f}")

# 2. Run One-Way ANOVA
# Method A: SciPy (Returns F-stat and p-value only)
f_stat, p_val = stats.f_oneway(*groups)
print(f"ANOVA F-stat: {f_stat:.4f}, p-value: {p_val:.6f}")

# Method B: Statsmodels (Returns full ANOVA table)
model = ols('MaxHR ~ C(ChestPain)', data=df).fit()
anova_table = anova_lm(model, typ=2)
print(anova_table)

3.7 Two-Way ANOVA

A Two-Way ANOVA is used when you have two categorical independent variables. This allows you to test the main effects of each variable, but more importantly, it tests the interaction effect between them.

Understanding Interaction Effects

An interaction occurs when the effect of one independent variable on the dependent variable depends on the level of the other independent variable.

Example: Does the effect of a Diet Plan on Cholesterol depend on the patient's Sex? If a diet works brilliantly for men but has no effect on women, that is a significant Diet × Sex interaction.

R Implementation
# The '*' operator includes both main effects and the interaction term
# Equivalent to: MaxHR ~ ChestPain + Sex + ChestPain:Sex
model_2way <- aov(MaxHR ~ ChestPain * Sex, data = heart)
summary(model_2way)

# Interpretation:
# 1. Look at the interaction term (ChestPain:Sex) first.
# 2. If interaction is significant (p < 0.05), the main effects cannot be interpreted in isolation.
# 3. If interaction is NOT significant, you can interpret the main effects of ChestPain and Sex.
Visualizing Interactions in R
library(ggplot2)
library(rstatix)

# Calculate means and standard errors for plotting
plot_data <- heart %>%
  group_by(ChestPain, Sex) %>%
  get_summary_stats(MaxHR, type = "mean_se")

ggplot(plot_data, aes(x = ChestPain, y = MaxHR, fill = Sex)) +
  geom_bar(stat = "identity", position = "dodge", color = "black") +
  geom_errorbar(aes(ymin = MaxHR - se, ymax = MaxHR + se), 
                position = position_dodge(0.9), width = 0.2) +
  labs(title = "Interaction Plot: MaxHR by Chest Pain and Sex",
       y = "Mean Maximum Heart Rate") +
  theme_minimal()

3.8 Repeated Measures ANOVA (Introduction)

Standard ANOVA assumes independence of observations. But what if you measure the same patients at multiple time points (e.g., Baseline, Week 4, Week 8)? The data points are correlated. Repeated Measures ANOVA handles this within-subjects design.

Key Concept: Partitioning Out Individual Differences

Because the same people are measured repeatedly, we can isolate the variance due to "Subject ID" and remove it from the error term. This dramatically increases statistical power.

The Sphericity Assumption

Repeated Measures ANOVA introduces a new assumption: Sphericity. This assumes that the variances of the differences between all possible pairs of within-subject conditions are equal.

  • Mauchly's Test: Tests for sphericity. If p < 0.05, sphericity is violated.
  • Correction: If violated, apply the Greenhouse-Geisser or Huynh-Feldt correction to the degrees of freedom.
R Implementation (Using ezANOVA)
library(ez)

# Data must be in "long" format: SubjectID, TimePoint, Cholesterol
# Example:
# Subject  Time   Chol
# 1        Pre    240
# 1        Post   210
# 2        Pre    230 ...

rm_anova <- ezANOVA(
  data = df_long,
  dv = Chol,             # Dependent Variable
  wid = SubjectID,       # Within-subject ID
  within = .(TimePoint), # The repeated factor
  detailed = TRUE
)

print(rm_anova$anova) # Look for Mauchly's Test and Greenhouse-Geisser p-value

3.9 Multiple Comparison Procedures (Post-Hoc Tests)

A significant ANOVA tells you that at least one group mean is different, but it doesn't tell you which ones. To find out, you must perform pairwise comparisons. However, doing multiple t-tests inflates the Type I error rate. Post-hoc tests correct for this.

1. Tukey's Honestly Significant Difference (HSD)

Tukey's HSD is the most common post-hoc test. It controls the Family-Wise Error Rate (FWER) at exactly α (e.g., 0.05) across all possible pairwise comparisons. It is highly recommended when you want to compare every group to every other group.

2. Bonferroni Adjustment

The Bonferroni correction is a simple, conservative method. You divide your desired overall α level by the number of comparisons you are making.

αadjusted = αoverall / m    (where m is the number of pairwise tests)

If you do 3 comparisons and want α=0.05, you only declare a result significant if its p-value is < 0.0167. It is very safe but can increase Type II errors (missing real differences).

R Implementation
# Assuming 'anova_result' is your significant One-Way ANOVA model

# Tukey's HSD
tukey_results <- TukeyHSD(anova_result, conf.level = 0.95)
print(tukey_results)
plot(tukey_results) # Visualize the confidence intervals

# Bonferroni Adjustment (using pairwise.t.test)
pairwise.t.test(heart$MaxHR, heart$ChestPain, p.adjust.method = "bonferroni")
Interpreting Tukey's Output

The output provides a p-value for every pair (e.g., typical vs asymptomatic). If the p adj < 0.05, those two specific groups have significantly different means. The 95% Family-Wise Confidence Interval that does not contain zero also indicates significance.

3.10 Effect Size

A p-value only tells you if an effect exists (statistical significance). It does not tell you if the effect is large enough to matter in the real world (clinical/practical significance). For this, we need Effect Sizes.

Eta-Squared (η2) for ANOVA

Eta-squared represents the proportion of the total variance in the dependent variable that is explained by the independent variable.

η2 = SSB / SST

Cohen's Guidelines for η2:

  • Small effect: 0.01
  • Medium effect: 0.06
  • Large effect: 0.14
Cramer's V for Chi-Square

For contingency tables, Cramer's V measures the strength of association between two categorical variables, ranging from 0 (no association) to 1 (perfect association).

V = √ [ χ2 / (n × df*) ]    (where df* = min(r-1, c-1))
R Implementation
# Eta-Squared for ANOVA
library(rstatix)
eta_squared(anova_result)

# Cramer's V for Chi-Square
library(rstatix)
# Assuming 'tab' is your contingency table
chisq_result <- chisq.test(tab)
cramer_v <- sqrt(chisq_result$statistic / (sum(tab) * min(nrow(tab)-1, ncol(tab)-1)))
print(cramer_v)

3.11 Practical Session: Comprehensive Workflow

Let's combine everything we've learned into a robust, reproducible statistical analysis pipeline using the Heart.csv dataset.

Scenario: Evaluating Diagnostic Markers and Treatment Efficacy

We will perform two analyses:

  1. Categorical: Is the Thalassemia stress test (Thal) associated with Heart Disease (AHD)?
  2. Continuous: Does Maximum Heart Rate (MaxHR) differ significantly across Chest Pain types, and which specific types differ?
Complete R Script
# ==============================================================================
# 1. SETUP AND DATA LOADING
# ==============================================================================
library(tidyverse)
library(car)       # For Levene's Test
library(rstatix)   # For effect sizes and post-hoc
library(ggpubr)    # For publication-ready plots

heart <- read.csv("Heart.csv") %>% na.omit()

# ==============================================================================
# 2. CATEGORICAL ANALYSIS: Chi-Square Test of Independence
# ==============================================================================
cat("\n--- CHI-SQUARE ANALYSIS ---\n")
tab <- table(heart$Thal, heart$AHD)
chi_res <- chisq.test(tab)

cat("Chi-Square Statistic:", chi_res$statistic, "\n")
cat("P-value:", chi_res$p.value, "\n")

# Calculate Cramer's V
n <- sum(tab)
df_star <- min(nrow(tab)-1, ncol(tab)-1)
cramer_v <- sqrt(chi_res$statistic / (n * df_star))
cat("Cramer's V (Effect Size):", cramer_v, "\n")

# Interpretation
if(chi_res$p.value < 0.05) {
  cat("Conclusion: Thal and AHD are significantly associated.\n")
} else {
  cat("Conclusion: No significant association found.\n")
}

# ==============================================================================
# 3. CONTINUOUS ANALYSIS: One-Way ANOVA
# ==============================================================================
cat("\n--- ONE-WAY ANOVA ANALYSIS ---\n")

# A. Check Assumptions
# 1. Normality of Residuals
model_aov <- aov(MaxHR ~ ChestPain, data = heart)
shapiro_p <- shapiro.test(residuals(model_aov))$p.value
cat("Shapiro-Wilk P-value (Normality):", shapiro_p, "\n")

# 2. Homogeneity of Variance
levene_res <- leveneTest(MaxHR ~ ChestPain, data = heart)
levene_p <- levene_res[1, "Pr(>F)"]
cat("Levene's P-value (Homogeneity):", levene_p, "\n")

# B. Run ANOVA
anova_sum <- summary(model_aov)
f_val <- anova_sum[[1]]$`F value`[1]
p_val <- anova_sum[[1]]$`Pr(>F)`[1]
cat("F-Statistic:", f_val, "| P-value:", p_val, "\n")

# C. Effect Size
eta_sq <- eta_squared(model_aov)
cat("Eta-Squared:", eta_sq$eta2[1], "\n")

# D. Post-Hoc Test (Tukey's HSD)
if(p_val < 0.05) {
  cat("\nPerforming Tukey's HSD...\n")
  tukey <- TukeyHSD(model_aov)
  print(tukey)
  
  # Visualize Post-Hoc Results
  plot(tukey, las = 1, col = "blue")
}
Complete Python Script
import pandas as pd
import numpy as np
import scipy.stats as stats
from statsmodels.formula.api import ols
from statsmodels.stats.anova import anova_lm
from statsmodels.stats.multicomp import pairwise_tukeyhsd

# 1. Load Data
df = pd.read_csv("Heart.csv").dropna()

# ==============================================================================
# 2. CATEGORICAL ANALYSIS: Chi-Square
# ==============================================================================
print("--- CHI-SQUARE ANALYSIS ---")
contingency = pd.crosstab(df['Thal'], df['AHD'])
chi2, p_chi, dof, expected = stats.chi2_contingency(contingency)

# Cramer's V
n = contingency.sum().sum()
v = np.sqrt(chi2 / (n * min(contingency.shape[0]-1, contingency.shape[1]-1)))

print(f"Chi2: {chi2:.4f}, P-value: {p_chi:.6f}")
print(f"Cramer's V: {v:.4f}")

# ==============================================================================
# 3. CONTINUOUS ANALYSIS: One-Way ANOVA
# ==============================================================================
print("\n--- ONE-WAY ANOVA ANALYSIS ---")

# A. Check Assumptions
groups = [group['MaxHR'].values for name, group in df.groupby('ChestPain')]

# Normality (Shapiro-Wilk on residuals)
model = ols('MaxHR ~ C(ChestPain)', data=df).fit()
residuals = model.resid
stat_shap, p_shap = stats.shapiro(residuals)
print(f"Shapiro-Wilk P-value: {p_shap:.6f}")

# Homogeneity of Variance (Levene's)
stat_lev, p_lev = stats.levene(*groups)
print(f"Levene's P-value: {p_lev:.6f}")

# B. Run ANOVA
anova_results = anova_lm(model, typ=2)
print("\nANOVA Table:")
print(anova_results)

# C. Effect Size (Eta-Squared)
ss_between = anova_results.loc['C(ChestPain)', 'sum_sq']
ss_total = anova_results['sum_sq'].sum()
eta_sq = ss_between / ss_total
print(f"\nEta-Squared: {eta_sq:.4f}")

# D. Post-Hoc Test (Tukey's HSD)
tukey = pairwise_tukeyhsd(endog=df['MaxHR'],    # Response variable
                          groups=df['ChestPain'], # Grouping variable
                          alpha=0.05)             # Significance level
print("\nTukey's HSD Results:")
print(tukey)

Exercise 1: Choosing the Right Test
Correct Answer:
  • a) One-Way ANOVA (Continuous DV, 1 categorical IV with >2 levels).
  • b) Chi-Square Test of Independence (Two categorical variables).
  • c) Chi-Square Goodness-of-Fit (One categorical variable compared to a theoretical distribution).
  • d) Paired t-test or Repeated Measures ANOVA (Same subjects measured twice).
Exercise 2: Interpreting ANOVA Output
Correct Answer:
  • a) Since p = 0.013 < 0.05, we reject the null hypothesis. There is a statistically significant difference in mean cholesterol levels among the three diets.
  • b) The degrees of freedom are dfB = 2, dfW = 87. Total N = dfB + dfW + 1 = 2 + 87 + 1 = 90 patients.
  • c) η2 = 0.09. According to Cohen's guidelines (0.01=small, 0.06=medium, 0.14=large), this represents a medium effect size. The type of diet explains 9% of the variance in cholesterol levels, which is a clinically meaningful impact.
Exercise 3: Assumption Violations
Correct Answer:
  • Violated Assumption: Homogeneity of Variance (Homoscedasticity). The p-value (0.002) is < 0.05, meaning the variances across groups are significantly unequal.
  • Normality: The Shapiro-Wilk p-value (0.15) is > 0.05, so the normality assumption is met.
  • Solution: You should use Welch's ANOVA, which does not assume equal variances. (Alternatively, a data transformation like Log or Square Root could be attempted, but Welch's is the direct and robust solution).
Exercise 4: Post-Hoc Interpretation
Correct Answer:
  • Significant Difference: Only Diet A and Diet B are significantly different from each other (p adj = 0.04 < 0.05).
  • Non-Significant Differences: Diet A vs Diet C, and Diet B vs Diet C are not significantly different.
  • Interpretation: Even though the overall ANOVA was significant (meaning not all diets are equal), Tukey's test reveals that the difference is driven entirely by the gap between Diet A and Diet B. Diet C's mean falls somewhere in between, and its difference from either A or B is not large enough to be statistically significant after correcting for multiple comparisons.
